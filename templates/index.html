How This Site Works

This dashboard is deployed on an AWS EC2 instance using:

  • Python + Flask for the backend
  • Gunicorn + systemd for service management
  • Nginx as a reverse proxy
  • SQLite for local persistence
  • JavaScript (AJAX) for dynamic dashboards
